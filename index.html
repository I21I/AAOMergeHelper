AAO Merge Helper

Avatar OptimizerのMergeSkinnedMeshとMergePhysBoneを簡単に設定するツールです。

VCC(ALCOM)で簡単インストール

下記のボタンをクリックするだけで、VCC(ALCOM)にリポジトリを追加できます。

VCC(ALCOM)に追加

補足：上記のボタンクリックでリポジトリの追加ができない場合は、以下の手動インストール手順をお試しください。ブラウザによっては「vcc://」から始まるリンクが正しく機能しないことがあります。

VCC手動インストール

  1. VCCを開き、「Settings」タブを選択
  2. 「Packages」タブをクリック
  3. 「Add Repository」ボタンをクリック
  4. 以下のURLを入力：
https://i21i.github.io/vpm-repos/vpm.json
  1. 「Add」ボタンをクリック
  2. 「I Understand, Add Repository」をクリックして確認
  3. 「Projects」タブを開き、「Manage Project」から「AAO Merge Helper」をインストール

ALCOM手動インストール

  1. ALCOMを開き、「パッケージ管理」タブを選択
  2. 「VPMリポジトリを追加」ボタンをクリック
  3. 以下のURLを入力：
https://i21i.github.io/vpm-repos/vpm.json
  1. 「VPMリポジトリを追加」ボタンをクリック
  2. 再度「VPMリポジトリを追加」ボタンをクリック
  3. 「プロジェクト」タブを開き、「管理」から「AAO Merge Helper」をインストール

機能

  • AAOのMergeSkinnedMeshとMergePhysBoneを自動でセット
  • オブジェクトのトグル制御のフィルタに対応(一部ツール)
  • 改変軽量化作業の効率化

動作環境

  • Unity 2022.3.22f1
  • VRCSDK3 3.8.1-beta.1
  • Avatar Optimizer 1.8.10
